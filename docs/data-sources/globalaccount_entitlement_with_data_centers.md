# btp_globalaccount_entitlement_with_data_centers (Data Source)

Returns entitlement details for a specific plan of a given service for a global account including the data center availability.

__Tip:__
You must be assigned to either the global account admin or global account viewers role.

## Example Usage

```terraform
data "btp_globalaccount_entitlement_with_data_centers" "by_name" {
	  service_name  = "hana-cloud"
	  plan_name     = "hana"
	}
```
